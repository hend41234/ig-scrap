# to get link post, run the ptr.js / ptr_extra.js
    
```bash
node ptr.js -u username -q 10 --sv output_name -t json
```
or

```bash
node ptr.js -u username -s yyyy-mm-dd --sv outpur -t json
```

for detail run :
```bash
node ptr.js --help
```
